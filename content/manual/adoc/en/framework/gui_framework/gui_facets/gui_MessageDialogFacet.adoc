`MessageDialogFacet` is a <<gui_facets,facet>> that provides an ability to pre-configure a <<gui_message_dialog,message dialog>>. Declarative definition of the message dialog replaces existing `Dialogs.createMessageDialog()` method. `MessageDialogFacet` is defined in the `facets` element of the screen XML descriptor.

Componentâ€™s XML-name: `messageDialog`.

----
include::{sourcesdir}/gui_facets/gui_MessageDialogFacet_example.xml[]
----

The screen configured with `MessageDialogFacet` can be shown explicitly using the `show()` method:

[source, java]
----
include::{sourcesdir}/gui_facets/gui_MessageDialogFacet_show.java[]
----
